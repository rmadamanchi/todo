### Create app folders

```bash
mkdir ~/src/github.com/<username>/todo
cd  ~/src/github.com/<username>/todo
```

### Hello World
Create `main.go`

```go
import "fmt"

func main() {
  fmt.Println("Hello World")
}
```

Run main.go
```bash
go run main.go
```
